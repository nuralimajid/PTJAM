Tag yang Harus Ada pada sebuah file html adalah

  • tag <!DOCTYPE html> = Tag ini untuk menginisiasi bahwa file ini berupa file html 5
  • tag <html> </html> = Tag ini adalah elemen root dari dokumen HTML dan mengelilingi semua elemen HTML lainnya.
  • tag <head> </head> = Tag ini berisi informasi meta tentang dokumen, seperti judul halaman, tautan ke stylesheet, dan meta tag atau biasa di sebut kepala bagian di html.
  • <body> </body> = Tag ini berisi semua konten yang akan ditampilkan di halaman web, seperti teks, gambar, tautan, dan lainnya. atau biasa di sebuy badan untuk menampung konten
